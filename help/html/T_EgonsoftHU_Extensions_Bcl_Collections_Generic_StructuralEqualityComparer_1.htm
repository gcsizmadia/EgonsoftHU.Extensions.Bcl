      Remarks

      With this comparer you can use types that implement IStructuralEquatable interface (e.g. arrays, value tuples) as the key in a dictionary.

      Example

      Using the constructor.
      var dictionary = new Dictionary<byte[], object>(StructuralEqualityComparer<byte[]>.Default);
      Using LINQ.
      var dictionary =
    collection.ToDictionary(
        element => element.ByteArrayProperty,
        element => element,
        StructuralEqualityComparer<byte[]>.Default
    );
